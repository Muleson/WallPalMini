## XML Representation (What Xcode Creates)

When you configure URL Types in Xcode, it adds this to your Info.plist:

```xml
<key>CFBundleURLTypes</key>
<array>
    <dict>
        <key>CFBundleTypeRole</key>
        <string>Editor</string>

        <key>CFBundleURLName</key>
        <string>com.samquested.grigrimvp</string>

        <key>CFBundleURLSchemes</key>
        <array>
            <string>crahg</string>
        </array>
    </dict>
</array>
```
